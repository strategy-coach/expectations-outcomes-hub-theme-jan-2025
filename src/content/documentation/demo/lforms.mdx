# LForms Demo

## lhc-forms-widget

lhc-forms-widget is an Astro component used to display LForms. Form definitions are stored in a JSON structure, which is either FHIR Questionnaire (recommended) or the LHC-Forms internal format. We created the component to render the form as HTML.

## Live Demo

[Try the LForms Demo](https://demo.hub.opsfolio.com/lforms)  


## Sample Forms and Data

In this demo, we have added some sample forms, and you can also view the data received from the submitted forms.
